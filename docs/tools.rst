Tools
=======
See: https://provis.readthedocs.org/en/latest/tools.html

Included in ``etc/`` are configuration files for:

* `Bash`_
*  Bash `bashmarks`
* `Compiz`_
* `Gnome`_
* `Htop`_
* `Git`_
* `Mercurial`_
* `Python 2`_
*  Python `Distribute`_
*  Python `Pip`_
*  Python `PDB`_
*  Python `IPDB`_
*  Python `IPython`_
*  Python `Virtualenv`_
*  Python `Virtualenvwrapper`_
* `Readline`_
* `Ruby`_
*  Ruby `Gems`_
* `Vim`_
*  Vim `NERDTree`_
* `Vimperator`_
* `ZSH`_
